¡Hola! Mi nombre es Francella, este es un proyecto de conciencia alimentaria. A veces no nos damos cuenta lo abundantes que somos y dejamos que nuestra comida se expire. Este proyecto nació de una necesidad que tenía como persona — utilicé tecnología para dejar una semillita de que esto se puede evitar y puede llegar a ser divertido y muy nutritivo. Lo iré mejorando poco a poco. 🌱
